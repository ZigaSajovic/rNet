# rNet
Batched RNN + LSTM in python and numpy

For an example of use consult the _char\_example.py_ file, where the model is used to train a character level text generator, as seen in the paper [Generating Sequences With Recurrent Neural Networks](https://arxiv.org/abs/1308.0850) by Alex Graves.
